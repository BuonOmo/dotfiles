# Dotfiles

Greatly inspired by https://www.atlassian.com/git/tutorials/dotfiles

Install with:

```zsh
zsh -c "$(curl -fsSL https://raw.githubusercontent.com/BuonOmo/dotfiles/main/install.zsh)"
```

Update with:

```zsh
dotf pull
```

Contribute with:

```zsh
dotf add .somerc
dotf commit
# fork if needed
dotf push
# PR welcome
```
